주요 내용 요약

Arjovsky 등은 GAN의 학습 불안정 문제를 이론적으로 분석하고, Jensen-Shannon 발산을 Wasserstein 거리(Earth Mover's Distance)로 대체한 WGAN을 제안하였다.

Wasserstein 거리는 두 분포의 겹침이 없어도 의미 있는 기울기를 제공하므로, 기존 GAN에서 발생하던 기울기 소실 문제를 해결한다. 판별자를 '크리틱(critic)'으로 재정의하고, 출력에 시그모이드를 적용하지 않는다.

WGAN은 GAN 학습의 안정성을 크게 개선하였으며, 크리틱 손실이 생성 품질과 상관관계를 가져 학습 진행을 모니터링할 수 있게 되었다. WGAN-GP(gradient penalty)로 발전하여 더욱 안정적인 학습이 가능해졌다.

주요 내용 요약

Ba, Kiros, Hinton은 배치 차원이 아닌 특징 차원에서 정규화를 수행하는 레이어 정규화(Layer Normalization)를 제안하였다. 배치 크기에 독립적이므로 시퀀스 길이가 다양한 RNN에서 특히 유용하다.

배치 정규화가 미니배치의 통계를 사용하여 배치 크기가 작으면 불안정해지는 반면, 레이어 정규화는 단일 샘플 내에서 정규화하므로 이 문제가 없다. 추론 시에도 학습 시와 동일한 연산을 사용하여 구현이 간단하다.

트랜스포머의 'Add & Norm' 구조에서 Norm은 레이어 정규화를 의미하며, 현대 LLM에서는 Pre-LN(서브레이어 앞에 배치) 구조가 표준이다.
